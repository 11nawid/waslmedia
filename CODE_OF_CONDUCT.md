# Code of Conduct

## Our standard

We want this project to be welcoming, practical, and respectful.

Examples of behavior that help:

- being respectful in feedback
- assuming good intent
- focusing on the code and the idea, not the person
- sharing context clearly and calmly

Examples of unacceptable behavior:

- harassment or personal attacks
- hate speech
- intimidation
- doxxing
- intentionally disruptive behavior

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.

## Reporting

If someone is making the project unsafe, contact the maintainer privately through the channel listed in [SECURITY.md](./SECURITY.md).
