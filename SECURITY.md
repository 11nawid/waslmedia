# Security Policy

## Report a vulnerability

If you discover a security issue in Waslmedia, please report it privately instead of opening a public GitHub issue.

## Private contact

- Instagram DM: [@11.skibidi](https://www.instagram.com/11.skibidi/)

## Helpful report details

- short summary of the issue
- affected route, feature, or file if known
- possible impact
- reproduction steps
- suggested fix if you have one

## Disclosure approach

- private reports are preferred until a fix or mitigation is ready
- share only the information needed to reproduce the issue safely
- responsible reports may be credited after the issue is resolved
