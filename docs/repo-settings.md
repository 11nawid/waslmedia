# Recommended GitHub Repo Settings

These are the recommended GitHub-side settings for this repository.

## General

- Set the repository description to mention video publishing, creator tools, Next.js, and ads platform workflows.
- Add topics such as `nextjs`, `react`, `typescript`, `video-platform`, `creator-tools`, `mysql`, `redis`, and `minio`.
- Keep the social preview image aligned with the Waslmedia brand.

## Security and analysis

- Enable Dependabot alerts
- Enable Dependabot security updates
- Enable secret scanning if available on the account plan
- Enable code scanning alerts so CodeQL results are visible

## Branch protection

- Protect `main`
- Require pull request reviews for direct changes
- Require status checks to pass before merge
- Block force pushes to `main`

## Community

- Keep issue templates and pull request templates enabled
- Point users to `SECURITY.md` for private vulnerability reporting
- Keep the license, code of conduct, and contributing guide visible
